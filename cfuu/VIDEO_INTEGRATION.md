# 视频攻略接入

## 当前方案

网站是纯静态部署，视频攻略读取 `public/data/video-guides.json`。页面提供：

- 抖音、快手关键词检索入口
- 平台与攻略类型筛选
- 按 `popularity` 降序展示收录视频
- 跳转平台原始视频页

新增视频时填写 `id`、`platform`、`title`、`author`、`url`、`category`、
`popularity` 和 `popularityLabel`。热度数据是维护时的快照，不是实时数据。

## 官方 API 升级

抖音和快手的视频列表接口需要开放平台应用、接口权限和用户 OAuth 授权，
并且主要读取已授权账号发布的视频。应用密钥不能放进 Vite 前端。

获得平台凭证后，应增加独立服务端任务：

1. 服务端保存 `client_secret` 和授权账号令牌。
2. 定时读取已授权创作者的视频列表与互动数据。
3. 按悠悠牌关键词过滤并生成 `video-guides.json`。
4. 静态站继续读取同一 JSON，无需重写页面组件。

官方接口不能直接匿名检索全站“CFM 悠悠牌”热门视频，因此不要在前端抓取
抖音或快手网页。
